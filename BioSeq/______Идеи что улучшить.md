- [SPARQL](https://sparql.uniprot.org/)
**SPARQL** (произносится "спаркл") — это **язык запросов** к базе данных, аналог SQL, но для так называемых "связанных данных" (RDF/семантический веб). UniProt хранит всю информацию в виде графа связей (белок → ген → организм → функция → болезнь и т.д.), и через SPARQL можно писать сложные запросы вроде "покажи мне все белки человека, которые связаны с мембраной, участвуют в апоптозе и имеют известную 3D-структуру". Это инструмент для **продвинутых пользователей** — биоинформатиков, которые хотят выкачивать большие наборы данных по сложным критериям. Обычному пользователю он почти никогда не нужен.

Можно чтоб нейросеть составляла SPARQL запросы для Uniprot
