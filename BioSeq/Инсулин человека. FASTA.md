# Инсулин человека в формате [[FASTA]]

Пример записи белка в FASTA:

```fasta
>sp|P01308|INS_HUMAN Insulin OS=Homo sapiens OX=9606 GN=INS PE=1 SV=1
MALWMRLLPLLALLALWGPDPAAAFVNQHLCGSHLVEALYLVCGERGFFYTPKT
RREAEDLQVGQVELGGGPGAGSLQPLALEGSLQKRGIVEQCCTSICSLYQLENYCN
```

## Разбор заголовка

В FASTA первая строка начинается с `>` и называется заголовком. В ней записана информация о последовательности, а ниже идет сама аминокислотная или нуклеотидная последовательность.

| Часть заголовка | Что означает |
|---|---|
| `>` | Признак начала FASTA-заголовка. |
| `sp` | База [[_UniProt|UniProtKB]]/[[Swiss-Prot]]: запись проверена и аннотирована вручную. |
| `P01308` | [[Accession number]]: стабильный идентификатор записи в UniProt. |
| `INS_HUMAN` | Entry name: короткое имя записи. `INS` — инсулин, `HUMAN` — человек. |
| `Insulin` | Название белка. |
| `OS=Homo sapiens` | Organism Species: вид организма, здесь человек. |
| `OX=9606` | Organism Taxonomy ID: таксономический идентификатор человека в [[NCBI Taxonomy]]. |
| `GN=INS` | Gene Name: имя гена, кодирующего белок инсулин. |
| `PE=1` | [[Protein Existence]]: уровень доказательности существования белка. `1` значит, что есть данные на уровне белка. |
| `SV=1` | Sequence Version: версия аминокислотной последовательности. |

## Другой стиль заголовка

NCBI-style может выглядеть примерно так:

```fasta
>NP_000198.1 insulin [Homo sapiens]
MALWMRLLPLLALLALWGPDPAAA
```

Здесь `NP_000198.1` — идентификатор белковой последовательности в [[NCBI]] [[RefSeq]], `insulin` — название белка, а `[Homo sapiens]` — организм.
