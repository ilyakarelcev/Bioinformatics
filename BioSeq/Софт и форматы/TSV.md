# TSV и CSV

**TSV** (Tab-Separated Values) — это простой текстовый формат для хранения таблиц. В нем значения в строке разделяются символом табуляции, а каждая новая строка файла соответствует новой строке таблицы.

Пример TSV:

```tsv
id	name	score
1	Anna	95
2	Ivan	88
```

**CSV** (Comma-Separated Values) — похожий текстовый формат, но значения чаще всего разделяются запятыми. Его удобно открывать в Excel, Google Sheets, Python, R и других инструментах для анализа данных.

Пример CSV:

```csv
id,name,score
1,Anna,95
2,Ivan,88
```

Главное отличие: в TSV разделитель — табуляция, а в CSV — запятая. Если внутри данных часто встречаются запятые, TSV может быть удобнее.
