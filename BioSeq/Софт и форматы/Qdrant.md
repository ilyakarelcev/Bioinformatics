Qdrant - это уже **vector database**. То есть не просто библиотека, а сервер с API.

**Qdrant** — (от англ. Quantum DRive ANalytics Technology) — это ==высокопроизводительная векторная база данных и поисковый движок с открытым исходным кодом, написанный на языке Rust==. Он предназначен для эффективного хранения, поиска и управления высокоразмерными векторами (эмбеддингами), обеспечивая семантический поиск (по смыслу, а не ключевым словам) для изображений, текстов и аудио.


```
id: "P12345"
vector: [...]
payload: {
  "uniprot_id": "P12345",
  "organism": "Homo sapiens",
  "reviewed": true,
  "gene": "TP53"
}

```
