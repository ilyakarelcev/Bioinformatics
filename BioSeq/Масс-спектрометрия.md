# [[Трипсин]]

**Трипсин** — это фермент, то есть белковая “молекулярная ножница”.
Он режет другие белки в довольно предсказуемых местах: обычно **после аминокислот лизин K и аргинин R**.
А [[Пептид]] после трипсина часто имеет примерно 7–25 аминокислот

```
Белок:
M-A-L-Q-V-K-G-T-P-R-S-L-F...

Трипсин режет после K и R:

M-A-L-Q-V-K | G-T-P-R | S-L-F...
```

Есть и другие “ножницы”:

```
трипсин — после K/R  
химотрипсин — рядом с ароматическими аминокислотами F/Y/W  
Lys-C — после K  
Glu-C — после E, иногда D  
Asp-N — перед D
```
Их используют, когда нужно получить другие куски или лучше покрыть белок.

## **[[Bottom-up proteomics]]** — это подход “снизу вверх”.
```
Белок целиком  
↓ режем трипсином  
Пептиды  
↓ масс-спектрометрия  
Массы и фрагменты пептидов  
↓ сравнение с базой данных  
Понимаем: ага, это был белок X
```
bottom-up проще, стабильнее, массово используется чаще.
## [[Top-down proteomics]]
```
Белок целиком
   ↓ масс-спектрометрия
Масса целого белка + его фрагментация
   ↓ анализ
Понимаем, что это за белок и какие у него модификации
```
