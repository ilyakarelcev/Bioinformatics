**ColBERT** — это более продвинутый метод поиска и reranking’а.

Название примерно от:
**Contextualized Late Interaction over BERT**

Главная идея: вместо того чтобы сжимать весь текст в один embedding, ColBERT хранит более детальные token-level embedding’и и сравнивает запрос с документом более тонко.

Проще:

Обычный embedding-поиск:

```
весь запрос → один вектор
весь документ → один вектор
сравнили два вектора
```

ColBERT:

```
запрос → много векторов по токенам
документ → много векторов по токенам
сравнили более детально
```

Обычно это точнее, но тяжелее по памяти и вычислениям.
