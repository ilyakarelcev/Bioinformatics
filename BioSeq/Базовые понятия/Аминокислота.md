Аминокислоты - строительные блоки белков. В белковых последовательностях их часто записывают однобуквенными кодами: например `M`, `K`, `T`, `F`.

В белках обычно встречают 20 стандартных аминокислот. У каждой есть общий аминокислотный "скелет" и своя боковая цепь, которая влияет на форму белка, растворимость, заряд, гибкость и химические взаимодействия.

## Стандартные аминокислоты

| Аминокислота                           | Код      | Коротко                                            |
| -------------------------------------- | -------- | -------------------------------------------------- |
| [[Аланин]]                             | `A`, Ala | маленькая неполярная аминокислота                  |
| [[Аминокислоты/Аргинин]]               | `R`, Arg | положительно заряженная аминокислота               |
| [[Аминокислоты/Аспарагин]]             | `N`, Asn | полярная незаряженная аминокислота                 |
| [[Аминокислоты/Аспарагиновая кислота]] | `D`, Asp | отрицательно заряженная аминокислота               |
| [[Аминокислоты/Цистеин]]               | `C`, Cys | может образовывать дисульфидные связи              |
| [[Аминокислоты/Глутамин]]              | `Q`, Gln | полярная незаряженная аминокислота                 |
| [[Аминокислоты/Глутаминовая кислота]]  | `E`, Glu | отрицательно заряженная аминокислота               |
| [[Аминокислоты/Глицин]]                | `G`, Gly | самая маленькая и гибкая аминокислота              |
| [[Аминокислоты/Гистидин]]              | `H`, His | часто участвует в активных центрах ферментов       |
| [[Аминокислоты/Изолейцин]]             | `I`, Ile | гидрофобная разветвленная аминокислота             |
| [[Лейцин]]                             | `L`, Leu | гидрофобная разветвленная аминокислота             |
| [[Аминокислоты/Лизин]]                 | `K`, Lys | положительно заряженная аминокислота               |
| [[Аминокислоты/Метионин]]              | `M`, Met | часто стартовая аминокислота при трансляции        |
| [[Аминокислоты/Фенилаланин]]           | `F`, Phe | ароматическая гидрофобная аминокислота             |
| [[Аминокислоты/Пролин]]                | `P`, Pro | ограничивает гибкость цепи и создает изгибы        |
| [[Аминокислоты/Серин]]                 | `S`, Ser | полярная аминокислота с гидроксильной группой      |
| [[Аминокислоты/Треонин]]               | `T`, Thr | полярная аминокислота с гидроксильной группой      |
| [[Аминокислоты/Триптофан]]             | `W`, Trp | крупная ароматическая аминокислота                 |
| [[Аминокислоты/Тирозин]]               | `Y`, Tyr | ароматическая аминокислота с гидроксильной группой |
| [[Валин]]                              | `V`, Val | гидрофобная разветвленная аминокислота             |
